# ezyvet demo

0. Run
> Run `index.php` with parameter as below.

1. Show cart
> List all item in cart
```$xslt
index.php?add=show
```  

2. Add to cart
> Add the item to cart by item name. (Session)
```$xslt
index.php?add=Sledgehammer
```  

2. Empty cart
> Delete all item from cart
```$xslt
index.php?add=clear
```
